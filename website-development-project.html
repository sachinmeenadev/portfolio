Project Name : PF-AIMS
Client : Poornima University, Jaipur
Completion : Sep 2017

This is an ERP developed to manage all the work of this institution in a central system that provides a flexible and remote access to all the staff members. This is a single web application with different authorization level so that each end user has the same app install and according to their authorization level, they all get different views. This also reduced the development cost and time.
